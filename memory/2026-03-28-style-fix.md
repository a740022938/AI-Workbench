**关键文件**: `dataset_export_window.py` → `dataset_exporter.py`  
**终点**: 导出数据集到 `dataset_output/`，生成报告

### 5. 闭环修正主链
**起点**: `main_window.py` → 闭环修正按钮 或 训练结果回流  
**关键文件**: `closed_loop_window.py` → `closed_loop_manager.py` → `badcase_manager.py`  
**终点**: 问题状态更新，建议回流到质检或训练

### 6. AI动作调用链
**起点**: OpenClaw命令或代码调用  
**关键文件**: `action_system.py` → `openclaw_bridge.py` → 各业务模块  
**终点**: 结构化动作回执和状态更新

### 7. 跨模块联动链
**起点**: 任一核心模块发现问题  
**关键文件**: 各模块管理器 → `action_system.py` 统一协调  
**终点**: 问题在对应模块得到处理，状态同步更新
